<h1>Shadeup</h1>

</center>

Shadeup is a CLI tool for scaffolding various Unreal Engine shader patterns.

# Installation & Usage

```sh
$ npm install shadeup
```

```sh
$ cd MyUnrealProject
$ shadeup # This will start the scaffolding wizard
```

# See [shadeup.dev](https://shadeup.dev/docs) for usage
